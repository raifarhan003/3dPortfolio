# Rai Farhan - Portfolio Website 🚀

This repository contains the source code for my personal 3D portfolio website.  
Feel free to explore the code to learn more about how it was built!

---

## ⚙️ Tech Stack

This project was built using modern web technologies:
- **React** & **TypeScript**
- **Three.js**, **React Three Fiber** & **Drei** for 3D rendering
- **GSAP** for advanced smooth animations
- **Vite** for fast bundling

---

## 🛠️ Setup Instructions

To run this project locally on your machine:

1. **Clone the repository:**
   ```bash
   git clone https://github.com/raifarhan003/RaiFarhanPortfolio.git
   cd RaiFarhanPortfolio
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Start the development server:**
   ```bash
   npm run dev
   ```

4. **Build for production:**
   ```bash
   npm run build
   ```
